# eye-check-v4
【2026计算机毕设项目】眼底眼疾识别系统~Python+深度学习+人工智能+算法模型

## 一、介绍
眼疾识别系统，（['白内障(Cataract)', '糖尿病视网膜病变(Diabetic retinopathy)', '青光眼(Glaucoma)', '正常(Normal)']），技术栈如下：
● 前端：Vue3+Element plus
● 后端：Flask框架
● 算法：卷积神经网络算法（resnet50）、TensorFlow

关键技术：深度学习、模型训练、图像识别、人工智能、Python、CNN卷积网络
